## Installation
Add this line to your application's Gemfile:

```ruby
gem 'xlsx_renderer'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install xlsx_renderer
```

Optionaly, you can run the generator, which create initializer file.

```bash
$ rails g xlsx_renderer:initializer
```
